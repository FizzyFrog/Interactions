к’·пё¶к’·к’Ҙк’·вҖ§ рқҷёрқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқрқҡ’рқҡҳрқҡ—рқҡң вӮҠЛҡк’·пё¶к’·к’Ҙк’·вҖ§вӮҠЛҡ

  В»В»вҖ”вҖ”вҖ”вҖ”> рқҷёрқҷјрқҷҝрқҷҫрқҡҒрқҡғрқҷ°рқҷҪрқҡғ <вҖ”вҖ”вҖ”вҖ”В«В«

вқҘ : рқҷёрқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқрқҡ’рқҡҳрқҡ—рқҡң рқҡҶрқҡ’рқҡқрқҡ‘ рқҷјрқҡҺ вқў ~ рҹҰӯ

рҹҗҫ : рқҷөрқҡӣрқҡ’рқҡҺрқҡ—рқҡҚрқҡ•рқҡў вҸӨвҳҶ 
рқҷё рқҡҠрқҡ– рқҡҳрқҡ”рқҡҠрқҡў рқҡ рқҡ’рқҡқрқҡ‘ рқҡҠрқҡ—рқҡўрқҡҳрқҡ—рқҡҺ рқҡ’рқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқрқҡ’рқҡ—рқҡҗ рқҡңрқҡҳ рқҡ•рқҡҳрқҡ—рқҡҗ рқҡҠрқҡң рқҡқрқҡ‘рқҡҺрқҡў рқҡ рқҡҺрқҡӣрқҡҺ рқҡ—рқҡҳрқҡқ рқҡңрқҡқрқҡҠрқҡқрқҡҺрқҡҚ рқҡ’рқҡ— рқҡ–рқҡў рқҷірқҷҪрқҷё, рқҷё'рқҡ– рқҡҹрқҡҺрқҡӣрқҡў рқҡҸрқҡӣрқҡ’рқҡҺрқҡ—рқҡҚрқҡ•рқҡў рқҡҠрқҡң рқҡ•рқҡҳрқҡ—рқҡҗ рқҡҠрқҡң рқҡқрқҡ‘рқҡҺ рқҡ”рқҡ’рқҡ—рқҡҚрқҡ—рқҡҺрқҡңрқҡң рқҡ’рқҡң рқҡ–рқҡһрқҡқрқҡһрқҡҠрқҡ•! 

вҸӨвң© рҹҰў : рқҡҒрқҡҳрқҡ•рқҡҺрқҡҷрқҡ•рқҡҠрқҡў-рқҡ’рқҡңрқҡ‘
рқҷё рқҡҷрқҡӣрқҡҺрқҡҸрқҡҺрқҡӣ рқҡқрқҡҳ рқҡҳрқҡ—рқҡ•рқҡў рқҡ‘рқҡҠрқҡҹрқҡҺ рқҡӣрқҡҳрқҡ•рқҡҺрқҡҷрқҡ•рқҡҠрқҡўрқҡң рқҡ рқҡ’рқҡқрқҡ‘ рқҡҳрқҡ—рқҡҺ рқҡңрқҡҷрқҡҺрқҡҢрқҡ’рқҡҸрқҡ’рқҡҢ рқҡҷрқҡҠрқҡӣрқҡқрқҡ—рқҡҺрқҡӣ рқҡҷрқҡҺрқҡӣ рқҡҢрқҡҳрқҡң. рқҷё рқҡ–рқҡҠрқҡў рқҡ‘рқҡҠрқҡҹрқҡҺ рқҡ–рқҡҳрқҡӣрқҡҺ рқҡқрқҡ‘рқҡҠрқҡ— рқҡҳрқҡ—рқҡҺ рқҡҷрқҡҺрқҡӣ рқҡҷрқҡҺрқҡӣрқҡңрқҡҳрқҡ— рқҡ’рқҡҸ рқҷё рқҡҸрқҡ’рқҡ—рқҡҚ рқҷё рқҡ•рқҡ’рқҡ”рқҡҺ рқҡқрқҡ‘рқҡҺрқҡ– рқҡҺрқҡ—рқҡҳрқҡһрқҡҗрқҡ‘. рқҷёрқҡҚ рқҡӣрқҡҠрқҡқрқҡ‘рқҡҺрқҡӣ рқҡқрқҡ‘рқҡ’рқҡң рқҡӣрқҡҺрқҡңрқҡҷрқҡҺрқҡҢрқҡқ рқҡӢрқҡҺ рқҡ–рқҡһрқҡқрқҡһрқҡҠрқҡ• рқҡӢрқҡһрқҡқ рқҡ’рқҡқ рқҡ’рқҡң рқҡ—рқҡҳрқҡқ рқҡӣрқҡҺрқҡҡрқҡһрқҡ’рқҡӣрқҡҺрқҡҚ, рқҡҳрқҡӢрқҡҹрқҡ’рқҡҳрқҡһрқҡңрқҡ•рқҡў, рқҡ‘рқҡҳрқҡ рқҡҺрқҡҹрқҡҺрқҡӣ, рқҷё'рқҡ•рқҡ• рқҡҷрқҡӣрқҡҳрқҡӢрқҡҠрқҡӢрқҡ•рқҡў рқҡӢрқҡҺ рқҡ•рқҡҺрқҡңрқҡң рқҡҺрқҡҠрқҡҗрқҡҺрқҡӣ. 


вқў ~ рҹҚҰ рқҷөрқҡ’рқҡЈрқҡЈ/рқҷөрқҡӣрқҡҳрқҡҗ - рқҹ·рқҹҪ - рқҷ¶рқҡҠрқҡў : КҡЙһ 

вқҘ : рқҷёрқҡ–рқҡҷрқҡҳрқҡӣрқҡқрқҡҠрқҡ—рқҡқ рқҷёрқҡ—рқҡҸрқҡҳ рқҷұрқҡҺрқҡ•рқҡҳрқҡ  вқў ~ рҹҰҲ

# рҹҡЁ : рқҗғрқҗҚрқҗҲ , рқҗҲрқҗҚрқҗ“ , рқҗҒрқҗҳрқҗҲ

рҹ«Ҹ ~ вҡ  : рқҗғрқҗҚрқҗҲ р“Ҷ© вқў р“ҶӘ рқҷ°рқҡ—рқҡҗрқҡҺрқҡ•рқҷјрқҡҳрқҡқрқҡ‘ , рқҷјрқҡҠрқҡ–рқҡ–рқҡҳрқҡ— рқҡЎ рқҷөрқҡ’рқҡЈрқҡЈ (рқҷөрқҡ’рқҡЈрқҡЈрқҡ–рқҡҳрқҡ—), рқҡ…рқҡҳрқҡЎ рқҡЎ рқҷІрқҡ‘рқҡҠрқҡӣрқҡ•рқҡ’рқҡҺ, рқҡ…рқҡҺрқҡҺрқҡң рқҡңрқҡһрқҡҷрқҡҷрқҡҳрқҡӣрқҡқрқҡҺрқҡӣрқҡң, рқҷҝрқҡӣрқҡҳрқҡңрқҡ‘рқҡ’рқҡҷрқҡҷрқҡҺрқҡӣрқҡң / рқҡғрқҡӣрқҡҠрқҡ—рқҡңрқҷёрқҷі / "рқҷҙрқҡ—рқҡҚрқҡҳрқҡҗрқҡҺрқҡ—рқҡ’рқҡҢ", <рқҹ·рқҹј рқҡҸрқҡҳрқҡӣ рқҷІ+рқҷ·, <рқҹ·рқҹә рқҷёрқҡ— рқҡҗрқҡҺрқҡ—рқҡҺрқҡӣрқҡҠрқҡ•

рҹ’ҡ : рқҗҲрқҗҚрқҗ“ р“Ҷ© вқў р“ҶӘ рқҷөрқҡ’рқҡЈрқҡЈрқҡ–рқҡҳрқҡҚрқҡҺрқҡһрқҡң, рқҡӮрқҡқрқҡҠрқҡқрқҡ’рқҡҢрқҷјрқҡҳрқҡқрқҡ‘, рқҷ·рқҡһрқҡңрқҡ”рқҡҺрқҡӣрқҡҚрқҡһрқҡңрқҡқ, рқҡӮрқҡқрқҡҳрқҡ•рқҡ’рқҡқрқҡЈ, рқҡӮрқҡқрқҡҠрқҡӣрқҷҫрқҡ рқҡ• /рқҷҝ (рқҡ…рқҡҠрқҡңрқҡңрқҡҠрқҡҗрқҡҳ & рқҡӮрқҡқрқҡҳрқҡ•рқҡҠрқҡң), рқҷұрқҡ•рқҡҳрқҡҳрқҡ—рқҡҠ /рқҷҝ (рқҷ»рқҡҳрқҡҳрқҡ—рқҡҠ & рқҷұрқҡ•рқҡ’рқҡқрқҡЈ) ~ Кҡрҹ§ЎЙһ

рҹҺӯ : рқҗҒрқҗҳрқҗҲ р“Ҷ© вқў р“ҶӘ рқҷё рқҡӣрқҡҺрқҡҢрқҡ’рқҡҷрқҡӣрқҡҳрқҡҢрқҡҠрқҡқрқҡҺ рқҡҺрқҡ—рқҡҺрқҡӣрқҡҗрқҡў! рқҷё рқҡҠрқҡ– рқҡҹрқҡҺрқҡӣрқҡў рқҡҺрқҡЎрқҡҷрқҡӣрқҡҺрқҡңрқҡңрқҡ’рқҡҹрқҡҺ рқҡ рқҡ’рқҡқрқҡ‘ рқҡ–рқҡў рқҡҠрқҡҢрқҡқрқҡ’рқҡҳрқҡ—рқҡң, рқҡ’рқҡҸ рқҡңрқҡҳрқҡ–рқҡҺрқҡҳрқҡ—рқҡҺ рқҡӢрқҡҳрқҡқрқҡ‘рқҡҺрқҡӣрқҡң рқҡ–рқҡҺ рқҡҠрқҡ—рқҡҚ рқҡ’рқҡқ'рқҡң рқҡ—рқҡҳрқҡқ рқҡҺрқҡ—рқҡҳрқҡһрқҡҗрқҡ‘ рқҡҸрқҡҳрқҡӣ рқҡҠ рқҡ—рқҡҺрқҡҺрқҡҚ рқҡқрқҡҳ рқҡңрқҡҷрқҡҺрқҡҠрқҡ”, рқҡ–рқҡў рқҡҠрқҡҢрқҡқрқҡ’рқҡҳрқҡ—рқҡң рқҡ рқҡ’рқҡ•рқҡ• рқҡҺрқҡЎрқҡҷрқҡӣрқҡҺрқҡңрқҡң рқҡ’рқҡқ рқҡ–рқҡҳрқҡӣрқҡҺ рқҡқрқҡ‘рқҡҠрқҡ— рқҡ–рқҡў рқҡ рқҡҳрқҡӣрқҡҚрқҡң, рқҡ’рқҡҸ рқҡқрқҡ‘рқҡ’рқҡң рқҡӢрқҡҳрқҡқрқҡ‘рқҡҺрқҡӣрқҡң рқҡўрқҡҳрқҡһ рқҡ’рқҡҚрқҡ” рқҡ рқҡ‘рқҡҠрқҡқ рқҡқрқҡҳ рқҡқрқҡҺрқҡ•рқҡ• рқҡһ рқҡ‘рқҡҳрқҡ—рқҡҺрқҡңрқҡқрқҡ•рқҡў! рқҷё'рқҡ– рқҡ”рқҡ’рқҡ—рқҡҚ рқҡ’рқҡҸ рқҡһрқҡӣ рқҡ”рқҡ’рқҡ—рқҡҚ рқҡқрқҡҳ рқҡ–рқҡҺ рқҡқрқҡ‘рқҡҳрқҡһрқҡҗрқҡ‘!
