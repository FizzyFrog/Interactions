### к’·пё¶к’·к’Ҙк’·вҖ§ рқҷёрқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқрқҡ’рқҡҳрқҡ—рқҡң вӮҠЛҡк’·пё¶к’·к’Ҙк’·вҖ§вӮҠЛҡ

г…Ө   В»В»вҖ”вҖ”вҖ”вҖ”> рқҷёрқҷјрқҷҝрқҷҫрқҡҒрқҡғрқҷ°рқҷҪрқҡғ <вҖ”вҖ”вҖ”вҖ”В«В«


вқҘ : рқҷёрқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқрқҡ’рқҡҳрқҡ—рқҡң рқҡҶрқҡ’рқҡқрқҡ‘ рқҷјрқҡҺ вқў ~ рҹҰӯ

рҹҗҫ : рқҷөрқҡӣрқҡ’рқҡҺрқҡ—рқҡҚрқҡ•рқҡў вҸӨвҳҶ 
рқҷё рқҡҠрқҡ– рқҡҳрқҡ”рқҡҠрқҡў рқҡ рқҡ’рқҡқрқҡ‘ рқҡҠрқҡ—рқҡўрқҡҳрқҡ—рқҡҺ рқҡ’рқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқрқҡ’рқҡ—рқҡҗ рқҡңрқҡҳ рқҡ•рқҡҳрқҡ—рқҡҗ рқҡҠрқҡң рқҡқрқҡ‘рқҡҺрқҡў рқҡ рқҡҺрқҡӣрқҡҺ рқҡ—рқҡҳрқҡқ рқҡңрқҡқрқҡҠрқҡқрқҡҺрқҡҚ рқҡ’рқҡ— рқҡ–рқҡў рқҷірқҷҪрқҷё, рқҷё'рқҡ– рқҡҹрқҡҺрқҡӣрқҡў рқҡҸрқҡӣрқҡ’рқҡҺрқҡ—рқҡҚрқҡ•рқҡў рқҡҠрқҡң рқҡ•рқҡҳрқҡ—рқҡҗ рқҡҠрқҡң рқҡқрқҡ‘рқҡҺ рқҡ”рқҡ’рқҡ—рқҡҚрқҡ—рқҡҺрқҡңрқҡң рқҡ’рқҡң рқҡ–рқҡһрқҡқрқҡһрқҡҠрқҡ•! 

вҸӨвң© рқҡҒрқҡҳрқҡ•рқҡҺрқҡҷрқҡ•рқҡҠрқҡў-рқҡ’рқҡңрқҡ‘ : рҹҰў
рқҷё рқҡҷрқҡӣрқҡҺрқҡҸрқҡҺрқҡӣ рқҡқрқҡҳ рқҡҳрқҡ—рқҡ•рқҡў рқҡ‘рқҡҠрқҡҹрқҡҺ "рқҡӣрқҡҳрқҡ•рқҡҺрқҡҷрқҡ•рқҡҠрқҡўрқҡң" (рқҡҢрқҡһрқҡҚрқҡҚрқҡ•рқҡ’рқҡ—рқҡҗ + рқҡ‘рқҡ’рқҡҚрқҡ’рқҡ—рқҡҗ, рқҡӣрқҡҺрқҡ•рқҡҠрқҡЎрқҡ’рқҡ—рқҡҗ) рқҡ рқҡ’рқҡқрқҡ‘ рқҡҳрқҡ—рқҡҺ рқҡңрқҡҷрқҡҺрқҡҢрқҡ’рқҡҸрқҡ’рқҡҢ рқҡҷрқҡҠрқҡӣрқҡқрқҡ—рқҡҺрқҡӣ рқҡҷрқҡҺрқҡӣ рқҡҢрқҡҳрқҡң. рқҷё рқҡ–рқҡҠрқҡў рқҡ‘рқҡҠрқҡҹрқҡҺ рқҡ–рқҡҳрқҡӣрқҡҺ рқҡқрқҡ‘рқҡҠрқҡ— рқҡҳрқҡ—рқҡҺ рқҡҷрқҡҺрқҡӣ рқҡҷрқҡҺрқҡӣрқҡңрқҡҳрқҡ— рқҡ’рқҡҸ рқҷё рқҡҸрқҡ’рқҡ—рқҡҚ рқҷё рқҡ•рқҡ’рқҡ”рқҡҺ рқҡқрқҡ‘рқҡҺрқҡ– рқҡҺрқҡ—рқҡҳрқҡһрқҡҗрқҡ‘. рқҷёрқҡҚ рқҡӣрқҡҠрқҡқрқҡ‘рқҡҺрқҡӣ рқҡқрқҡ‘рқҡ’рқҡң рқҡӣрқҡҺрқҡңрқҡҷрқҡҺрқҡҢрқҡқ рқҡӢрқҡҺ рқҡ–рқҡһрқҡқрқҡһрқҡҠрқҡ• рқҡӢрқҡһрқҡқ рқҡ’рқҡқ рқҡ’рқҡң рқҡ—рқҡҳрқҡқ рқҡӣрқҡҺрқҡҡрқҡһрқҡ’рқҡӣрқҡҺрқҡҚ, рқҡҳрқҡӢрқҡҹрқҡ’рқҡҳрқҡһрқҡңрқҡ•рқҡў. рқҡӮрқҡҳрқҡ–рқҡҺрқҡқрқҡ’рқҡ–рқҡҺрқҡң рқҷё рқҡ–рқҡҠрқҡў рқҡ“рқҡң рқҡӢрқҡҺ рқҡҢрқҡһрқҡҚрқҡҚрқҡ•рқҡ’рқҡ—рқҡҗ рқҡ / рқҡңрқҡҳрқҡ–рқҡҺрқҡҳрқҡ—рқҡҺ рқҡҠрқҡқ рқҡӣрқҡҠрқҡ—рқҡҚрқҡҳрқҡ– рқҡҠрқҡ—рқҡҚ рқҡ’рқҡқ рқҡ—рқҡҳрқҡқ рқҡ–рқҡҺрқҡҠрқҡ— рқҡҠрқҡ—рқҡўрқҡқрқҡ‘рқҡ’рқҡ—рқҡҗ рқҡқрқҡҳрқҡҳ. 

рҹ§Ӣ : рқҷҪрқҡҳрқҡқрқҡҺ вҸӨвҳҶ 
рқҷё рқҡҚрқҡҳрқҡ—рқҡқ рқҡҢрқҡҠрқҡӣрқҡҺ рқҡ’рқҡҸ рқҡўрқҡҳрқҡһ рқҡңрқҡ‘рқҡ’рқҡҷ рқҡқрқҡ‘рқҡҺ рқҡңрқҡ‘рқҡ’рқҡҷрқҡң рқҷё рқҡ‘рқҡҠрқҡҹрқҡҺ рқҡҠрқҡң рқҡҠ рқҡҚрқҡ—рқҡ’, рқҡңрқҡ‘рқҡ’рқҡҷ рқҡ рқҡқрқҡҹ рқҡһ рқҡ рқҡҠрқҡ—рқҡқ! рқҡғрқҡ‘рқҡ’рқҡң рқҡҷрқҡҠрқҡҗрқҡҺ рқҡ’рқҡң рқҡңрқҡҷрқҡҺрқҡҢрқҡ’рқҡҸрқҡ’рқҡҢрқҡҠрқҡ•рқҡ•рқҡў рқҡҸрқҡҳрқҡӣ рқҡңрқҡ‘рқҡ’рқҡҷрқҡң рқҡ’рқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқрқҡ’рқҡҳрқҡ—! рқҷ° рқҷІрқҡ‘рқҡҠрқҡӣрқҡ•рқҡ’рқҡҺ рқҡҢрқҡҠрқҡ— рқҡңрқҡқрқҡ’рқҡ•рқҡ• рқҡ’рқҡ—рқҡқрқҡҺрқҡӣрқҡҠрқҡҢрқҡқ рқҡ’рқҡҸ рқҷё рқҡҠрқҡ– рқҡ…рқҡҳрқҡЎ рқҡ рқҡ’рқҡқрқҡ‘рқҡҳрқҡһрқҡқ рқҡ–рқҡҠрқҡ”рқҡ’рқҡ—рқҡҗ рқҡ’рқҡқ рқҡңрқҡҺрқҡҺрқҡ– рқҡ•рқҡ’рқҡ”рқҡҺ рқҡҠ рқҡңрқҡ‘рқҡ’рқҡҷ.
